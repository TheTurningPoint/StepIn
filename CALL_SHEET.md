## What it does — by who uses it

### Resident (their phone, their daily companion)
- **Recovery-day counter** — sober days, front and center.
- **Meeting check-in** — GPS location-stamped + a witness signs on the phone; counts toward the weekly
  requirement.
- **Curfew sign in / out** — "I'm leaving / I'm home," with where they're going; location captured.
- **Document signing** — house forms appear and they sign with a finger.
- **File a concern (grievance)** — can be anonymous.
- **Change their own PIN**; forgot it → self-reset by email, or the manager resets it.

### House manager (runs the house)
- **"Needs attention" home** — open grievances, screenings to send to the lab, docs to countersign,
  and curfew sign-ins that happened away from the house.
- **Live In/Out board** — who's home, who's out and where, who's late.
- **Log a screening** — urine, breathalyzer, or both, **a result for each**, observed or not, signed by
  staff + resident; split results (one neg / one pos) are flagged; track the lab result on urine.
- **Log an incident** — typed, described, signed, timestamped.
- **Manage residents** — add/edit, contact + emergency info, move-in & sobriety dates, discharge.
- **Chores, events, announcements** — assign and post; residents acknowledge.
- **Reports** — Attendance, Screening, Incidents, Grievances → one-tap **PDF** (NARR-ready, dated,
  signed) or **CSV** export.
- **Reset a resident's PIN**, send required documents, set their own email for reminders.

### Owner (one or many houses)
- **Multi-house roll-up** — every house's status, residents, and on-track numbers in one view; drill
  into any house.
- **Org settings & branding** — org name (shown on app, reports, emails), weekly meeting requirement,
  lab policy, each house's check-in location.
- **Managers** — add a house/manager, reset a manager's PIN, remove a manager.
- **Activity log** — tamper-evident record of who changed or removed what.
- **App errors** — owner-only view if anything technical ever breaks.

### Behind the scenes (say it only if asked)
- **Daily reminder emails** to residents (documents/announcements) and a **weekly summary email** to
  owners/managers (meetings, screenings, positives, late, incidents, grievances).
- **Your own branded web address** (`yourhouse.instepapp.com`); works on any phone, **nothing to
  install**.
- **Privacy:** each house/org's data is walled off and locked to signed-in users; drug-test & medical
  notes protected the same way.

## Fast answers to "can it…?"

| They ask | Answer |
| --- | --- |
| …work if a resident has no smartphone? | Yes — staff do everything on their behalf. Phone is convenience, not required. |
| …handle more than one house? | Yes — owner view rolls them all up; each manager sees only their house. |
| …prove meetings really happened? | GPS location-stamp + an in-person witness signature on each check-in. |
| …do breathalyzer *and* urine on one test? | Yes — separate result for each; disagreements flagged as a split. |
| …send screens to a lab? | Yes — track lab result on urine specimens; per-house lab policy. |
| …give me audit paperwork? | Every report prints to a dated, signed PDF, or exports to a spreadsheet. |
| …let residents report problems? | Yes — grievances, anonymous if they want, tracked to resolution. |
| …keep data private/safe? | Walled per house/org, locked to logged-in users. Nothing readable without signing in. |
| …get set up fast? | A web link, no install. I set up your house + first login; you're live the same day. |

## Call flow (60 seconds of structure)
1. **Open the manager demo**, hand-walk "Needs attention" → In/Out → Log a screening → a PDF report.
2. **Switch to owner** (if multi-house) → the roll-up.
3. **Show the resident view** → "this is their daily companion, not surveillance."
4. **Say the price**, then **stop talking**: *"If it looks useful, I can have your house live today for
   $99 a month — want me to get you started?"*
5. Silence is fine. Let them think.
